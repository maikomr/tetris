Para correr la aplicación abre una terminal o línea de comandos.
Luego sitúate en el folder del juego tetris.

Ejecuta el comando `meteor npm install` para instalar los paquetes necesarios de npm.

Ejecuta `meteor` para lanzar la aplicación.

Abre un navegador y ve a la dirección `localhost:3000` para interactuar con el juego.
